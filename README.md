
Destinado ao processamento digital de imagens utilizando ferramentas como OpenCV e métodos de visão computacional além dos convencionais

## Instalação
Rode esse comando no terminal do projeto
```bash
pip install cv2 pytesseract ultralytics numpy pandas pathlib yolov5
```

## Execução
Use o comando 
```bash
python <ARQUIVO.PY>
```
ARQUIVO.PY representa o arquivo de algoritmo de detecção de placas

Atualmente existem varias versões de algoritmos de detecção pois a cada tentativa e melhoria criamos versões e versões de arquivos até chegarmos na final.
Verifique a pasta D:\outros sistemas\Process-IMG\backup\Yolo_v5_Plate, nela existe as versoes dos algoritmos desenvolvidos, a ultima versão é previsao17.py
